# api-starter

A simple Node.js AP starter

# Start

Simple:

```
npm install
npm start
```

## Developing

With a file watcher (`nodemon`)

```
npm run start:dev
```

## Config

Easily customizable configuration.

With a `.env` file in the project root folder

```
#.env

PORT=3210
```

Or with environment variables

```
# Export a variable
export PORT=3210
npm start

# Directly in one command
PORT=3210 npm start
```
